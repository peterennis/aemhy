# nodemon

Run scripts in Node.JS using `nodemon`. Almost the same as `node` process with watcher.
- Same support as with the `node process`.
- Default: `./src`

## Default command
```bash
nodemon ./src
```

## CLI Options
https://github.com/remy/nodemon#nodemon

## Examples

### Default CLI
```bash
mhy nodemon
```

### Run a specific file
```bash
mhy nodemon file
```
